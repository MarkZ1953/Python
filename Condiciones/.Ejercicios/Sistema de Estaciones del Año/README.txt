Crear un programa en Python que solicite al usuario un número correspondiente a un mes del año y asigne la estación
del año correspondiente según la siguiente escala:

- 1, 2, 12: Invierno
- 3, 4, 5: Primavera
- 6, 7, 8: Verano
- 9, 10, 11: Otoño

El programa debe validar que el número ingresado por el usuario se encuentre dentro del rango válido de 1 a 12.

Finalmente imprimir el mes que el usuario ingreso y su estación correspondiente en la que se encuentra.
